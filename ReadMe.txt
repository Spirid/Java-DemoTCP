Task:
	Написать пример приложения, производящего обмен данными по протоколу TCP/IP.
 Клиент приложения должен реализовать следующий алгоритм работы:
	установить соединение с сервером приложения;
	последовательно послать серверу приложения строки «date» и «time». 
Сервер в ответ на первый запрос должен вернуть текущую дату в виде строки, например, «29.11.2012». 
На второй запрос сервер должен вернуть текущее время в виде строки, например, «11:46».
(Для получения числа, месяца, года, часа и минут можно воспользоваться классом java.util.GregorianCalendar);
	разорвать соединение с сервером.
